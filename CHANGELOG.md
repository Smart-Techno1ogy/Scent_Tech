# Changelog

## 3.1.0

- Renamed the highest fragrance preset from Strong to Intense.
- Added bounds validation before writing spray and pause settings.
- Expanded and redacted diagnostics for safer support reports.
- Added HACS installation, automation, protocol, and troubleshooting documentation.
- Added repository status badges and public-release notes.
- Removed `strings.json`; custom integrations load English strings from `translations/en.json`.
- Centralised manufacturer and model metadata used by entities.

## 3.0.1

- Prepared the project for HACS and public GitHub distribution.
- Added HACS and Hassfest validation workflows.
- Added repository documentation, licence, issue templates, and release metadata.

## 3.0.0

- Replaced the immediate power switch with a momentary Dispense now button.
- Renamed Schedule to Automatic diffusion.
- Added Light, Balanced, Strong, and Custom presets.
- Added square Home Assistant branding.

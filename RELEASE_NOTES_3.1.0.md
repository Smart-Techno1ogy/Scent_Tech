# Smart Technology Scent Diffuser v3.1.0

## Highlights

- Simple **Dispense now** action for an immediate three-second burst.
- **Automatic diffusion** with Light, Balanced, Intense, and Custom presets.
- Improved validation and privacy-safe diagnostics.
- Expanded HACS, automation, protocol, and troubleshooting documentation.

## Upgrade note

The highest preset is now named **Intense** instead of **Strong**. Automations that explicitly select `Strong` should be updated to select `Intense`. Existing entity IDs and unique IDs are unchanged.
